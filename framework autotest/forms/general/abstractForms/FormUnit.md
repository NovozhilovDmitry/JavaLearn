# FormUnit.java

## Описание
Базовый интерфейс для единиц формы, предоставляющий метод для возврата текущего экземпляра формы. Использует дженерики для типобезопасного возврата конкретного типа формы. Является расширением интерфейса FormWin и служит основой для других интерфейсов форм.

## Public методы
- `itSelf()` - возвращает текущий экземпляр формы с приведением к параметризованному типу

## Зависимости
- `forms.general.abstractForms.FormWin` - родительский интерфейс

## Пример использования
```java
public class МояФорма implements FormUnit<МояФорма> {
    // реализация методов формы
    public МояФформа какойтоМетод() {
        // какой-то код
        return itSelf(); // возвращает this с типом МояФорма
    }
}
```
