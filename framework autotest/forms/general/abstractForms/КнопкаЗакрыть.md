# КнопкаЗакрыть.java

## Описание
Интерфейс-миксин для форм, содержащих кнопку закрытия. Предоставляет стандартизированные методы для закрытия окон с различными сценариями перехода. Автоматически находит кнопку закрытия по стандартным атрибутам и инкапсулирует логику закрытия формы.

## Public методы
- `кнопкаЗакрыть()` - возвращает объект кнопки закрытия
- `закрыть(T existingClassInstance)` - закрывает окно с переходом на существующий экземпляр
- `закрыть(Class<T> new_Class_Instance)` - закрывает окно с переходом на новый экземпляр класса
- `закрыть(Class<T> new_Class_Instance, Object args)` - закрывает окно с переходом и параметрами
- `закрыть()` - закрывает окно без перехода

## Зависимости
- `forms.general.abstractForms.FormWin` - родительский интерфейс
- `enums.FindBy` - перечисление стратегий поиска
- `userInterface.Jump_Button` - UI-компонент кнопки

## Пример использования
```java
public class МояФорма implements КнопкаЗакрыть {
    // реализация FormWin
    public void закрытьФорму() {
        закрыть(СледующаяФорма.class, параметры);
    }
}
```

## Неясности
1. Разница между вариантами методов `закрыть` - когда использовать каждый сценарий
2. Обработка исключений при закрытии - что происходит при ошибках
