# Points.java

## Summary
Класс для работы с точками восстановления (restore points) в Oracle БД. Позволяет создавать, восстанавливать и удалять точки восстановления для тестовых целей. Автоматически генерирует имена точек по шаблону и управляет их жизненным циклом. Особенно полезен для отката состояния базы данных после выполнения тестов.

## Public методы
1. **создание_короткой_точки(String название_съюта)**  
   Создает точку восстановления с автоматически сгенерированным именем. Параметр: название_съюта - префикс для имени точки.

2. **восстановление_короткой_точки(@Nullable String часть_имени_точки)**  
   Восстанавливает состояние базы до указанной точки. Параметр: часть_имени_точки - если null, восстанавливает последнюю точку.

3. **восстановление_последней_короткой_точки()**  
   Восстанавливает последнюю созданную точку без указания имени.

4. **удаление_точек(String часть_названия, List<String> имена_точек)**  
   Удаляет точки по части имени или списку имен. Параметры: часть_названия - фильтр по имени, имена_точек - конкретные точки для удаления.

## Dependencies
- Oracle Database (v$restore_point, FLASHBACK TABLE)
- Внутренние утилиты: DateGenerator для работы с датами
- Projects enum (CREDIT, DEPOSIT)
- Table.execute, Table.getTable, Table.executeQueryData для SQL-запросов
- Environment.getDbProperties для получения параметров БД

## Usage Example
```java
// Создание точки восстановления
Points.создание_короткой_точки("MyTestSuite");

// Восстановление до конкретной точки
Points.восстановление_короткой_точки("MyTestSuite");

// Восстановление последней точки
Points.восстановление_последней_короткой_точки();
```

## Uncertainties
1. Логика определения "working" проектов закомментирована и не функционирует
2. Использование кириллических идентификаторов может вызвать проблемы с кодировкой
3. Ограничение в 1000 точек (1-999) может быть недостаточным
4. Нет обработки случаев, когда точка восстановления уже существует
5. Потенциальные проблемы с правами доступа к v$restore_point
6. Не ясна логика работы с временными метками в удалении устаревших точек
