# Пользователи.java

## Summary
Простой enum, представляющий список пользователей системы. Содержит перечень учетных записей, используемых в тестовых сценариях. Используется для идентификации пользователей при авторизации и выполнении операций с различными уровнями доступа.

## Public методы
Нет public методов, только значения перечисления.

## Dependencies
Нет внешних зависимостей.

## Usage Example
```java
// Использование в тестах авторизации
авторизоватьсяПод(Пользователи.user);

// Проверка прав доступа
if (текущийПользователь == Пользователи.oper_od) {
    // выполнение операций с повышенными правами
}
```

## Uncertainties
1. Назначение пользователя "viktor" не указано
2. Пользователь "oper_od" помечен как TODO для релиза 43, что может указывать на незавершенную функциональность
3. Отсутствуют дополнительные атрибуты пользователей (роли, права доступа)
