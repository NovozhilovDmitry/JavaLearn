# Расширения_файлов.java

## Summary
Enum для работы с файловыми расширениями в системе. Содержит основные расширения файлов, используемые в банковской системе, включая готовые (.ready) и архивные форматы. Предоставляет утилитные методы для определения расширения файла и проверки его наличия.

## Public методы
1. **getРасширение()**  
   Возвращает строковое значение расширения файла.

2. **haveExtensionOf(File file)**  
   Проверяет, имеет ли файл одно из известных расширений. Параметр: file - проверяемый файл.

3. **haveExtensionOf(String fName)**  
   Проверяет, имеет ли имя файла одно из известных расширений. Параметр: fName - имя файла.

4. **from(String fName)**  
   Определяет расширение файла по его имени. Возвращает Optional<Расширения_файлов>.

## Dependencies
- java.io.File
- java.util.Optional

## Usage Example
```java
// Проверка расширения файла
boolean isExcel = Расширения_файлов.haveExtensionOf("report.xls");

// Получение расширения
Optional<Расширения_файлов> ext = Расширения_файлов.from("data.xml");
if (ext.isPresent()) {
    // обработка XML файла
}

// Использование в фильтрации
File file = new File("document.txt");
if (Расширения_файлов.haveExtensionOf(file)) {
    // обработка текстового файла
}
```

## Uncertainties
1. Логика обработки расширения "empty" может быть неочевидной
2. Не все возможные расширения файлов покрыты enum'ом
3. Метод from() не обрабатывает файлы с несколькими точками в имени
