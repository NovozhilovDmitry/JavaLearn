# FileDKOeditor.java - Документация

## Summary
Интерфейс `FileDKOeditor` определяет контракт для редакторов файлов в системе ДКО. Он предоставляет методы для сохранения изменений в файлы как в исходный файл, так и в новый файл с указанным именем, а также для завершения редактирования без сохранения.

## Public методы
- **`save()`** - сохраняет изменения в исходный файл
- **`saveAs(String file_name)`** - сохраняет изменения в новый файл с указанным именем
- **`complete()`** - завершает редактирование без сохранения изменений

## Dependencies
- **`base.FileChecker`** - базовый интерфейс для работы с файлами
- **Не имеет внешних зависимостей**

## Usage Example
```java
FileDKOeditor<MyFileChecker> editor = getFileEditor();
editor.saveAs("modified_file.xml");
editor.complete();
```

## Uncertainties
1. **Различие между save и complete** - неясно, какие именно действия выполняются при complete()
2. **Обработка ошибок** - не определено поведение при ошибках сохранения
3. **Состояние редактора** - не указано, можно ли использовать редактор после complete()
