# NewXLS.java - Документация

## Summary
Класс `NewXLS` предоставляет функциональность для создания и выгрузки XLS-файлов в тестовом фреймворке. Он позволяет генерировать Excel-файлы с заданным содержимым и автоматически выгружать их в указанные шлюзы. Класс интегрирован с системой отчетов Allure для прикрепления созданных файлов и включает механизмы обработки ошибок копирования.

## Public методы
- **`NewXLS(Шлюзы FilePath, String FileName, String SheetName)`** - конструктор для создания файла в указанном шлюзе
- **`NewXLS(String SheetName, String FileName)`** - конструктор для создания файла во временной директории
- **`NewXLS(String SheetName, String FileName, String FilePath)`** - конструктор с полным контролем пути
- **`flushFile(Object[][] obj)`** - создает и выгружает файл с данными, копируя во временную директорию и целевой шлюз
- **`createBook(Object[][] obj)`** - создает книгу Excel с переданными данными и возвращает путь к файлу

## Dependencies
- **`jxl.*`** - библиотека Java Excel API для работы с XLS-файлами
- **`enums.Шлюзы`** - перечисление шлюзов для выгрузки файлов
- **`base.MainClass`** - основной класс фреймворка
- **`env.Environment`** - доступ к настройкам окружения
- **Внешние**: `java.io.*`, `java.nio.file.*`, `org.slf4j.Logger`

## Usage Example
```java
NewXLS xls = new NewXLS(Шлюзы.ШЛЮЗ_ПИС, "test_file", "Лист1");
Object[][] data = {
    {"Заголовок1", "Заголовок2"},
    {"Данные1", "Данные2"}
};
xls.flushFile(data);
```

## Uncertainties
1. **Форматирование ячеек** - используется только текстовый формат без поддержки других типов данных
2. **Обработка ошибок записи** - исключения WriteException обрабатываются только выводом в консоль
3. **Кодировка файлов** - не указана явная кодировка для текстового содержимого
4. **Временные файлы** - отсутствует механизм очистки временных файлов после выгрузки
