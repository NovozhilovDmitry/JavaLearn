# XlsChecker.java

## Summary
Устаревший класс для проверки данных в XLS-файлах. Осуществляет поиск файлов по шаблону имени и шлюзам, проверяет соответствие данных ожидаемым значениям в виде массивов или блоков. Поддерживает гибкую настройку проверяемых строк и колонок, включая работу с диапазонами и нестрогое сравнение.

## Public методы
1. **XlsChecker(String FileNamePart, String SheetName, Шлюзы... шлюзы)**  
   Конструктор - инициализирует проверщик с поиском файла по шаблону в указанных шлюзах.

2. **getFileName()**  
   Возвращает полный путь к найденному файлу.

3. **checkXslData(String reference, String expectedValue)**  
   Проверяет значение конкретной ячейки. Параметры: reference - адрес ячейки, expectedValue - ожидаемое значение.

4. **getRowCount()**  
   Возвращает количество строк в листе.

5. **setColumns(String letters)**  
   Задает колонки для проверки в формате "A,C-F". Поддерживает диапазоны.

6. **setIndexes(String indexes)**  
   Задает строки для проверки в формате "1,3-6". Поддерживает диапазоны.

7. **checkByArray(Object[][] array, String[][]... blocks)**  
   Основной метод проверки данных по массиву и блокам. Может вызывать Assertion.fail() при несоответствиях.

8. **получитьИндексаНайденныхСтрок(String[][] ПервыеИзСдвоенныхСтрок)**  
   Находит индексы строк, содержащих заданные данные. Возвращает ArrayList<ArrayList<Integer>>.

9. **проверитьСвязанныеСтрокиЭксель(String[][] ПервыеИзСдвоенныхСтрок, String[][] ДанныеСвязанныхСтрок)**  
   Проверяет связность строк (предполагается, что вторые строки следуют за первыми).

10. **setStrict(boolean Strict)**  
    Переключает режим строгого сравнения (полное соответствие) / нестрогого (содержание).

11. **deleteFile()**  
    Вызывает очистку скачанных форм (autoClearDownloadedForms).

12. **найтиЗаданныеПодстроки(String[] stringsForFind, int[] rows)**  
    Ищет подстроки в указанных строках, требует единичного вхождения каждой.

13. **найтиЗаданнуюСтрокуИзНесколькихЯчеек(String[][] stringsForFind, int[] rows)**  
    Аналогично предыдущему, но для строк из нескольких ячеек.

14. **addBlock(String[][] строки)**  
    Добавляет блок данных для проверки.

15. **addBlocks(String[][][] блоки)**  
    Добавляет несколько блоков данных.

## Dependencies
- Apache POI (HSSFWorkbook, HSSFSheet) - работа с Excel
- SLF4J - логирование
- Allure Framework (@Step)
- Внутренние утилиты: FileSearcher, DateGenerator
- JUnit 5 (Assertions.fail)
- Кастомные enum: Шлюзы, Расширения_файлов

## Usage Example
```java
new XlsChecker("form777_", "Лист1", Шлюзы.ODBR)
    .setIndexes("1,3")
    .setColumns("A,C")
    .addBlock(new String[][]{{"t1", "t2"}, {"", "t4"}})
    .checkByArray(new String[][]{{"text1", "text2"}, {"", "text3"}});
```

## Uncertainties
1. Назначение MainClass и метод autoClearDownloadedForms() не определены в коде
2. Логика преобразования чисел и дат в преобразованиеСтроки() может быть хрупкой
3. Неясна точная логика сравнения блоков в checkByArray()
4. Использование кириллических идентификаторов усложняет чтение кода
5. Назначение метода closeExcel() не реализовано (закомментировано)
6. Потенциальные проблемы с кодировками в файлах
