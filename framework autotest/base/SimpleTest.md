# SimpleTest.java - Документация

## Summary
Абстрактный класс `SimpleTest` предоставляет упрощенную базовую функциональность для тестов, не требующих сложной подготовки сценария. Он сочетает функциональность MainClass для работы с АРМ и AbstractTest для управления тестовыми данными. Класс предназначен для простых тестовых случаев с минимальной инфраструктурой.

## Public методы
- **`эталонныйФайл(String... subDirsAndFileName)`** - статический метод для получения пути к эталонному файлу в каталоге тестовых данных

## Dependencies
- **`base.MainClass`** - основной класс для работы с АРМ
- **`base.AbstractTest`** - интерфейс для работы с тестовыми данными
- **Внешние**: `java.nio.file.Paths`

## Usage Example
```java
public class SimplePaymentTest extends SimpleTest {
    @Test
    public void testSimplePayment() {
        String referenceFile = эталонныйФайл("payment_template.xml");
        // Использование эталонного файла в тесте
    }
}
```

## Uncertainties
1. **Ограниченная функциональность** - отсутствуют методы подготовки/очистки сценария
2. **Наследование от MainClass** - наследует всю сложность MainClass для простых тестов
3. **Отсутствие инициализации** - нет автоматической инициализации АРМ как в Scenario
