# Предустановленные_фильтры.java

## Описание
Класс для работы с формой "Предустановленные фильтры" в GUI-автотестах. Представляет окно со списком предустановленных фильтров и кнопкой закрытия. Реализует интерфейс FormWin для стандартизации работы с оконными формами.

## Public методы
- `закрыть()` - нажимает кнопку "Закрыть" для закрытия формы
- `win()` - возвращает объект окна (реализация интерфейса FormWin)

## Зависимости
- `base.MainClass` - базовый класс автотестов
- `enums.FindBy` - перечисление стратегий поиска элементов
- `forms.general.abstractForms.FormWin` - интерфейс оконных форм
- `userInterface.Jump_Button`, `userInterface.List`, `userInterface.Window` - UI-компоненты

## Пример использования
```java
Предустановленные_фильтры фильтры = new Предустановленные_фильтры();
фильтры.лист.выбрать_значение("Фильтр по дате");
фильтры.закрыть();
```

## Неясности
1. Назначение класса `asdco.ARM_KO.FrmPreInstaledFilter` - специфичный GUI-компонент
2. Логика работы списка `лист` - какие операции доступны с предустановленными фильтрами
3. Отсутствие методов для выбора фильтров из списка
