# Результат_проверки_обеспеченности_на_дату_ОД.java

## Описание
Класс для работы с формой отображения результатов проверки обеспеченности на определенную дату. Содержит таблицы с основной информацией и негодным обеспечением, поддерживает контекстное меню и операции расширения портфеля. Использует паттерн Singleton.

## Public методы
- `Результат_проверки_обеспеченности_на_дату_ОД(String date)` - конструктор с датой проверки
- `getInstance()` - возвращает экземпляр класса (Singleton)
- `основная_таблица()`, `таблица_негодное_обеспечение()` - возвращают таблицы
- `вывести_негодное_обеспечение()` - нажимает кнопку вывода негодного обеспечения
- `нажать_ПКМ_в_блоке_Протокол_обработки_и_выбрать_из_контекстного_меню(ContextMenuValues)` - работа с контекстным меню
- `нажать_Расширить_портфель(Class<T>, Object)` - нажатие кнопки расширения портфеля

## Зависимости
- `com.rational.test.ft.*` - Rational Functional Tester
- `forms.general.abstractForms.КнопкаЗакрыть`, `Таблица` - абстрактные формы
- `userInterface.*` - UI-компоненты

## Пример использования
```java
Результат_проверки_обеспеченности_на_дату_ОД результат = 
    new Результат_проверки_обеспеченности_на_дату_ОД("01.01.2024");
результат.вывести_негодное_обеспечение();
результат.нажать_ПКМ_в_блоке_Протокол_обработки_и_выбрать_из_контекстного_меню(ContextMenuValues.Копировать);
```

## Неясности
1. Назначение закомментированного элемента `протокол_обработки`
2. Логика работы метода `нажать_Расширить_портфель` - что происходит при расширении портфеля
3. Разделение на два класса (Objects и основной) - архитектурное решение
